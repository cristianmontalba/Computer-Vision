# Computer Vision

This repository contains assignments and evaluation codes developed as part of the **Computer Vision** course taught by **Prof. Domingo Mery (PUC)** during the **second semester of 2025**.  
The work focuses on fundamental and applied computer vision methods, including image processing, feature extraction, and pattern recognition.
